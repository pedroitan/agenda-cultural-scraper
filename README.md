# Agenda Cultural Salvador — Scraper

## Setup

1. Copie `.env.example` para `.env` e preencha `SUPABASE_URL` e `SUPABASE_SERVICE_ROLE_KEY`.
2. Instale dependências:

```bash
npm install
```

## Rodar local

```bash
npm run dev
```

## Notas

- Este scraper é `scraper-first` (MVP). O endpoint do Sympla pode mudar.
- Métricas são registradas em `scrape_runs`.
